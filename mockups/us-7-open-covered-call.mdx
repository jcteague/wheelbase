# US-7: Open a Covered Call

_As a wheel trader holding shares after CSP assignment, I want to sell a covered call against my shares and see whether the strike protects my cost basis, so that I can collect additional premium and know upfront whether I'd profit or lock in a loss if the shares are called away._

## Acceptance Criteria States

- **form**: Sheet open with default values; strike $182.00, expiration "2026-02-21", premium $2.30, 1 contract; above-cost-basis info note shown
- **below-cost-basis**: Strike $174.00 entered (below $176.50 cost basis) — gold warning "This strike is below your cost basis — you would lock in a loss of $2.50/share if called away"; Confirm button remains enabled
- **at-cost-basis**: Strike $176.50 entered (equal to cost basis) — gold warning "This strike is at your cost basis — you would break even if called away"
- **above-cost-basis**: Strike $182.00 entered — info note "Shares called away at $182.00 → profit of $5.50/share"
- **success**: Hero card "HOLDING 100 SHARES · CC OPEN"; result summary showing CC_OPEN leg, updated cost basis $174.20/share, total premium collected $460.00

// ── Guardrail logic ───────────────────────────────────────────────────────────
// Client-side only: strike vs basisPerShare comparison
// Real usage: inline in OpenCoveredCallSheet component
export function computeGuardrail(strikeStr, basisStr, sharesHeld) {
  const strike = parseFloat(strikeStr)
  const basis  = parseFloat(basisStr)
  if (isNaN(strike) || isNaN(basis)) return null
  const diff = strike - basis
  if (diff < 0) {
    return {
      type: 'below',
      message: `This strike is below your cost basis — you would lock in a loss of $${Math.abs(diff).toFixed(2)}/share if called away`,
    }
  }
  if (diff === 0) {
    return {
      type: 'at',
      message: 'This strike is at your cost basis — you would break even if called away',
    }
  }
  return {
    type: 'above',
    message: `Shares called away at $${strike.toFixed(2)} → profit of $${diff.toFixed(2)}/share`,
  }
}
