# US-12: Holding Shares Section on Positions List

_As a wheel trader viewing my positions list, I want HOLDING_SHARES positions displayed in their own section with share-relevant columns (Shares, Days Held, Premium Collected, Cost Basis), so that I see meaningful data instead of stale option fields like negative DTE._

## Acceptance Criteria States

- **with-holding-shares**: Three sections visible — "Active" (CSP_OPEN positions with option columns), "Holding Shares" (HOLDING_SHARES positions with share-relevant columns), and "Closed" (completed positions at reduced opacity)
- **no-holding-shares**: Only "Active" and "Closed" sections visible — "Holding Shares" section is hidden when no positions are in that phase

export function fmtMoney(v) {
  return (
    '$' + Number(v).toLocaleString('en-US', { minimumFractionDigits: 2, maximumFractionDigits: 2 })
  )
}
