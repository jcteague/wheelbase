# US-11: Wheel Leg Chain Display

_As a wheel trader reviewing an active or completed position, I want to see every leg in the wheel in chronological order alongside the running cost basis at each step, so that I can understand the complete history of the position and how each action contributed to my current or final cost basis._

## Acceptance Criteria States

- **partial**: Single CSP_OPEN row; running basis shows initial basis $176.50; no footer
- **mid-cycle**: Three rows (CSP_OPEN → ASSIGN → CC_OPEN); ASSIGN row shows "— (assigned)" + "100 shares received" annotation; CC_OPEN running basis drops to $174.20
- **complete**: Five rows through called-away; CC_CLOSE premium shows "−$1.80" in amber (buyback debit); CC_CLOSE running basis carries forward at $174.20; CALLED_AWAY shows "100 shares called away"; final P&L footer row "+$780.00" in green
- **cc-expired**: Four rows (CSP_OPEN → ASSIGN → CC_OPEN → CC_EXPIRED); CC_EXPIRED premium shows "expired worthless" in muted style; running basis carries forward $174.20

// Matches display logic described in US-11 technical notes:
// - ASSIGN: "— (assigned)" + "{n} shares received" annotation
// - CALLED_AWAY: "— (assigned)" + "{n} shares called away" annotation
// - CC_EXPIRED: "expired worthless" in muted italic
// - CC_CLOSE: "−$X.XX" in amber (buyback debit, not a credit — do not use green)
// - All others: "+$X.XX" in green

export function PremiumCell({ leg }) {
  if (leg.legRole === 'ASSIGN') {
    return (
      <span style={{ display: 'flex', flexDirection: 'column', gap: 2 }}>
        <span style={{ color: 'var(--wb-text-muted, #4a5a6a)', fontStyle: 'italic' }}>
          — (assigned)
        </span>
        <span style={{ color: 'var(--wb-text-muted, #4a5a6a)', fontSize: '0.68rem' }}>
          {leg.contracts * 100} shares received
        </span>
      </span>
    )
  }
  if (leg.legRole === 'CALLED_AWAY') {
    return (
      <span style={{ display: 'flex', flexDirection: 'column', gap: 2 }}>
        <span style={{ color: 'var(--wb-text-muted, #4a5a6a)', fontStyle: 'italic' }}>
          — (assigned)
        </span>
        <span style={{ color: 'var(--wb-text-muted, #4a5a6a)', fontSize: '0.68rem' }}>
          {leg.contracts * 100} shares called away
        </span>
      </span>
    )
  }
  if (leg.legRole === 'CC_EXPIRED' || leg.action === 'expire') {
    return (
      <span style={{ color: 'var(--wb-text-muted, #4a5a6a)', fontStyle: 'italic' }}>
        expired worthless
      </span>
    )
  }
  if (leg.legRole === 'CC_CLOSE') {
    return <span style={{ color: 'var(--wb-gold, #e6a817)' }}>−${leg.premiumPerContract}</span>
  }
  if (!leg.premiumPerContract)
    return <span style={{ color: 'var(--wb-text-muted, #4a5a6a)' }}>—</span>
  return <span style={{ color: 'var(--wb-green, #3fb950)' }}>+${leg.premiumPerContract}</span>
}

export function BasisCell({ value }) {
  if (!value) return <span style={{ color: 'var(--wb-text-muted, #4a5a6a)' }}>—</span>
  return <span style={{ color: '#79c0ff', fontWeight: 600 }}>${value}</span>
}
